# diagrams-net-azure-libraries

This is a collection of icon libraries for [Diagrams.net](https://app.diagrams.net/) (formerly known as Draw.io). These libraries were created leveraging the [Azure Icon Collection](https://code.benco.io/icon-collection/) maintained by [benc-uk](https://github.com/benc-uk)

## Ownership & Copyright

I do not attribute ownership to any of these icons & images. No copyright infringement is intended.  
All files have been sourced from the [Azure Icon Collection](https://code.benco.io/icon-collection/) which in turn were collected by scrapping the public internet and various Microsoft sites. They are included here under fair use.
